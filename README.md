# Магия внутри

Статический лендинг курса по материалам вводной презентации.

## Структура

- `index.html` - продающая страница
- `styles.css` - стили
- `assets/` - изображения, извлеченные из презентации

## Публикация

Сайт можно подключить к GitHub Pages, Netlify, Vercel или любому статическому хостингу. Точка входа: `index.html`.

Кнопка записи сейчас ведет на `mailto:hello@example.com`. После появления реальной ссылки курса замените адрес в секции `#join`.
